# Passo a passo para executar o desafio

1 - realizar o build

```
docker build -t williamkoller/codeeducation .
```

2 - rodar o docker

```
docker run -t williamkoller/codeeducation
```

3 - fazer o push da imagem

```
docker push williamkoller/codeeducation
```
